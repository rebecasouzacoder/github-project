<h2 align="center">Como Usar 🤔</h2>

```
- Clone esse repositório:
$ git clone https://github.com/rebecasouzacoder/github-project.git

- Entre no diretório:
$ cd  github-project

- Instale as dependências:
$ npm i

- Inicie o app:
$ npm run start
```
